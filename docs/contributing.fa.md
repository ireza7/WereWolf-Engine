# مشارکت

از اینکه به فکر مشارکت در پروژهٔ WereWolf Engine هستید سپاسگزاریم!

## راه‌های کمک

- **گزارش باگ:** یک Issue در گیت‌هاب باز کنید.
- **پیشنهاد ویژگی:** بحثی را در بخش Issues شروع کنید.
- **افزودن نقش جدید:** یک کلاس نقش بسازید و به‌عنوان افزونه به اشتراک بگذارید (به بخش [نقش‌ها](roles.md) مراجعه کنید).
- **بهبود تست‌ها:** تست‌هایی برای سناریوهای مرزی اضافه کنید.

## راه‌اندازی محیط توسعه

۱. مخزن را Fork کنید.
۲. Fork خود را clone کنید:

```bash
git clone https://github.com/your-username/WereWolf-Engine.git
cd WereWolf-Engine
```

۳. یک محیط مجازی بسازید و وابستگی‌های توسعه را نصب کنید:

```bash
python -m venv venv
source venv/bin/activate  # یا venv\Scripts\activate در ویندوز
pip install -e ".[dev]"
```

۴. تست‌ها را اجرا کنید تا از درستی همه چیز مطمئن شوید:

```bash
pytest
```

## راهنمای Pull Request

- یک شاخه برای تغییرات خود بسازید.
- اگر قابلیت جدیدی اضافه می‌کنید، حتماً تست بنویسید.
- مطمئن شوید همهٔ تست‌ها پاس می‌شوند (`pytest`).
- در صورت نیاز مستندات را به‌روز کنید.
- یک PR با توضیح واضح ارسال کنید.

## سبک کدنویسی

این پروژه از `black` و `ruff` برای قالب‌بندی یکدست کد استفاده می‌کند. قبل از commit آن‌ها را اجرا کنید:

```bash
black werewolf_engine tests
ruff check werewolf_engine tests
```

مشارکت‌های شما تحت همان مجوز MIT پروژه منتشر خواهد شد.
